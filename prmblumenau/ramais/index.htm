pelo telefone fixo na PRM:
- 2000 + matrícula = chamada para o Zoom daquela pessoa
- número local (sem código de área) = chamada local
- 0 + XX + número com código de área = chamada a distância
- *8 = capturar ligação de ramal do colega
- ** + ramal = capturar ligação do ramal informado

pelo Zoom (roteadas pela central da PRM):
- 4703 + ramal = chamada telefônica para o ramal (Blumenau)
- 0 + número local = chamada telefônica para o número
- 0 + 0XX + número com código de área = chamada telefônica a distância
